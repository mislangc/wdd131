WDD 131 Project Site Plan

Site Name

Gamers Alley

Site Purpose

Help users around the world to discover different games of different genres and display information about the game they might be interested in.

Scenarios

1. Ronnie can't find a specific game he is looking for. He decides to go to the "Contact Us" page to fill up a form to suggest that missing game to the web owners to be part of the website.

2. Michael is looking for a game of a specific genre. He clicks on the filter button to show only the games that had the same genre as the filter.
